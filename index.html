HWT550 - DevLog 005

21 de Janeiro de 2026

The colored rooms will be removed, since I’m changing the room concept. Before, this idea was heavily inspired by VVVVVV, but I realized that colored rooms no longer make sense for the game. Now there will be only one room, which will be gray.

HWT550 - DevLog 005

21 de Janeiro de 2026

Changes to the game’s concept. The game will now focus on the player going through each room individually. It will still keep the idea of colored rooms, but it will no longer follow the room structure seen in VVVVVV. This change happened by chance, as I realized it would be more interesting to design the game this way instead of creating a single giant room.

HWT550 - DevLog 002

20 de Janeiro de 2026

We will now have a trampoline in the game. This will help the player reach places that they normally wouldn’t be able to access. It will require some testing to see if this mechanic works well for the game. I’m also planning to add an inverted gravity mechanic, similar to the one in VVVVVV, but I’m not sure yet if this will be a good idea.
